# Todo Project

> A simple HTML todo project

![demo](demo.gif)

## Built With

- CSS, HTML, JS
- Webhint, Hint, Stylelint
- Webpack

## Getting Started

To get a local copy up and running clone the repo and run

`npm install`

### Install

to build a the project into `./dist` folder run

`npm run build`

### Usage

to start the webpack-dev-server run

`npm run serve`
